# Share

Share is a secure, self-hosted file sharing tool for my personal devices.

I built it because I was tired of relying on WhatsApp just to transfer PDFs, notes, and files between my laptop, phone, and tablet. Rather than uploading files to a third-party service, Share lets me securely expose a directory from my machine and access it directly from my trusted devices.

## Goals

* Secure file sharing between my personal devices.
* Device-based authentication using public-key cryptography (no usernames or passwords).
* Simple CLI powered by Cobra.
* Interactive terminal experience built with Bubble Tea.
* Modern, responsive web interface for browsing and downloading files.
* QR code generation for quick device pairing.
* Support for uploads, downloads, and directory browsing.
* Graceful shutdown and reliable file transfers.
* Future support for secure sharing beyond the local network.
